# Jerumed Nexus ‚Äî IT Solutions for Healthcare

## Setup (IMPORTANTE ‚Äî leggi prima!)

### Windows:
1. Estrai lo ZIP in una cartella
2. Apri la cartella in VS Code
3. Doppio click su `setup.bat` (oppure apri il terminale e scrivi `setup.bat`)
4. Aspetta che finisce
5. Scrivi nel terminale: `npm run dev`
6. Apri `http://localhost:3000`

### Mac/Linux:
1. Estrai lo ZIP
2. Apri il terminale nella cartella
3. `chmod +x setup.sh && ./setup.sh`
4. `npm run dev`
5. Apri `http://localhost:3000`
